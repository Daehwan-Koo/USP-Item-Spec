USP 제품 스펙 리스트 – 사용자 도움말 (KOR)

이 애플리케이션은 USP 제품의 기본 정보와 클레임 데이터를 관리하고, 제품 간 비교 및 검색기능을 제공합니다.

1. 로그인 및 사용자 역할

로그인 페이지

애플리케이션을 처음 사용할 때는 비밀번호를 입력하여 로그인할 수 있습니다.

  • Master: 모든 기능
  • Admin (관리자): 제품 추가, 수정, 삭제, 검색, 비교 및 일부 DB 관리 기능
  • Viewer (열람 전용): 제품 목록과 상세 정보를 조회할 수 있으며, 데이터 수정이나 삭제 권한은 없습니다.

2. 메인 대시보드 – 제품 목록 및 검색기능

제품 목록 테이블

메인 페이지에서는 USP 제품의 목록이 나타납니다. 각 행에는 다음과 같은 정보가 표시됩니다:

  • 제품 유형: 제품의 제형 정보
  • Item Code, Item Name, Description, Unit Size, Color, Weight (mg): 기본 제품 정보
  • Info: 마우스를 올리면 클레임 정보 및 리마크 내용을 확인할 수 있습니다.
  • Actions: 'View', 'Edit', 'Delete' 버튼을 통해 제품 상세 조회, 수정, 삭제가 가능합니다.
  • Compare: 비교할 제품들의 체크박스를 선택하고 Compare 버튼을 누르면 비교페이지로 이동합니다.

검색 기능

제품 검색 폼을 통해 다양한 필터(예: Item Code, Name, Description, Weight 등)를 이용해 원하는 제품을 검색할 수 있습니다.

"Select Unit Size" 및 "Select Color" 버튼을 누르면 원하는 옵션을 선택할 수 있습니다.

클레임 검색 필드를 추가하여 제품의 클레임 관련 조건으로 필터링할 수 있습니다.

3. 제품 추가하기 (Add Product)

제품 추가 페이지 (Add)

관리자 및 마스터 권한 사용자만 접근할 수 있습니다.

제품정보 추가 양식은 제품정보 양식표에 따라 추가하시기 바랍니다.

4. 제품 수정하기 (Edit)

제품 수정 페이지 (Edit)

관리자 및 마스터 사용자만 접근 가능합니다.

기존 제품의 정보와 클레임 데이터를 수정할 수 있습니다.

원하는 정보를 수정한 후 "Save" 버튼을 클릭하여 변경 사항을 저장합니다.

5. 제품 상세 보기

View 버튼 - 제품 상세 페이지

제품의 기본 정보와 클레임 데이터가 표시됩니다.

표 형식으로 제품의 각 항목(아이템 코드, 이름, 설명, 단위, 색상, 무게, 용량, 비고, 클레임정보 등)을 확인할 수 있습니다.

6. 제품 검색 및 필터링

검색 폼 사용법 (메인 페이지 내 검색 영역)

다양한 필드(예: Item Code, Item Name, Description, Weight 등)로 제품을 검색할 수 있습니다.

수치 필터:
Weight 입력 시 "Weight Tolerance (%)"를 함께 지정하여 허용 오차 범위 내의 제품을 검색할 수 있으며, 기본 10% 범위로 설정되어 있습니다.

Unit Size 및 Color 선택:
"Select Unit Size" 버튼과 "Select Color" 버튼을 누르면 각각 창이 열려, 원하는 옵션을 선택 후 적용할 수 있습니다.

클레임 검색:
여러 개의 클레임 검색 필드를 추가할 수 있으며, 각 행에서 주요 주장, 설명, 농도, 단위, 허용 오차, 그리고 "Include(포함)" 또는 "Exclude(제외)" 옵션을 지정할 수 있습니다.
"Add Claim Search" 버튼을 통해 필드 행을 추가합니다.
"Add Vitamins & Minerals" 버튼을 누르면 원하는 성분을 체크하여 클레임 검색창에 추가할 수 있으며, 체크한 순서대로 추가됩니다.

Load Specifications:
"Load Specifications" 버튼을 클릭하면 창이 열리고, Item Code를 입력하면 해당 제품의 클레임 정보가 자동으로 검색 필드에 추가됩니다.
검색 후 "Search" 버튼을 누르면 필터 조건에 맞는 제품 목록이 업데이트되며, "Clear" 버튼을 누르면 검색 조건이 초기화됩니다.
서치버튼 옆에 결과 내 재검색(Re-search) 기능이 있어, 체크 후 검색하면 이전 검색 결과 내에서 다시 검색할 수 있습니다.

7. 제품 비교

비교 기능 (Compare)

메인 페이지에서 각 제품 우측의 "Compare" 체크박스들을 선택하고 Compare 버튼을 누르면 제품 비교페이지로 이동하여, 코드별 기본정보와 클레임 데이터를 비교할 수 있습니다.

비교페이지 상단에 Item Code를 입력하고 Add 버튼을 누르면 비교표에 해당 제품이 추가됩니다.

비교 페이지에서는 두 개의 표로 제품을 나란히 비교할 수 있습니다.

  • 기본 정보 테이블: 각 제품의 Item Code, Unit Size, Color, Weight, Dosage, Remark 정보를 확인할 수 있습니다.
  • 클레임 테이블: 각 제품의 클레임 데이터를 비교하여, 테스트 결과에 따라 색상(검정: Test O, 빨강: Test X 혹은 Input)으로 표시됩니다.

제품 삭제: 비교 목록에서 "Delete" 버튼을 이용하여 제품을 제거할 수 있습니다.
