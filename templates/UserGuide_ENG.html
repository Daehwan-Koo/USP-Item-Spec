USP Product Specification List - User Guide (ENG)

This application manages the basic information and claim data of USP products, providing product comparison and search functionalities.

1. Login & User Roles

Login Page

Users can log in by entering their password when using the application for the first time.

User Roles:

  • Master: Access to all features.
  • Admin (Administrator): Can add, edit, delete, search, compare products, and manage certain database functions.
  • Viewer (Read-only): Can view product lists and details but cannot modify or delete data.

Product List Table

The main page displays the list of USP products. Each row contains the following information:

  • Product Type: Product formulation details.
  • Item Code, Item Name, Description, Unit Size, Color, Weight (mg): Basic product information.
  • Info: Hovering over the row reveals claim information and remarks.
  • Actions: 'View,' 'Edit,' and 'Delete' buttons allow users to view, modify, or remove products.
  • Compare: Selecting checkboxes of products and clicking the 'Compare' button navigates to the comparison page.

Search Function

Users can filter products using various fields (e.g., Item Code, Name, Description, Weight, etc.).

Clicking "Select Unit Size" or "Select Color" opens selection options.

A claim search field enables filtering based on claim-related conditions.

3. Adding a Product

Add Product Page

Accessible only to Admin and Master users.

Users should follow the product information form when adding a new product.

4. Editing a Product

Edit Product Page

Accessible only to Admin and Master users.

Users can modify existing product details and claim data.

After making the necessary changes, click the "Save" button to apply the updates.

5. Viewing Product Details

View Button - Product Detail Page

Displays basic product information and claim data.

The table format includes various product details such as Item Code, Name, Description, Unit, Color, Weight, Capacity, Remarks, and Claim Information.

6. Product Search & Filtering

Search Form Usage (Search Area on Main Page)

Users can search products using various fields (e.g., Item Code, Item Name, Description, Weight, etc.).

Numeric Filters:
When entering Weight, users can specify a "Weight Tolerance (%)" to find products within a tolerance range (default: 10%).

Unit Size & Color Selection:
Clicking "Select Unit Size" or "Select Color" opens selection windows.

Claim Search:
Users can add multiple claim search fields and specify conditions for each (e.g., main claim, description, concentration, unit, tolerance, and inclusion/exclusion criteria).
Clicking "Add Claim Search" adds a new row for claims.
Clicking "Add Vitamins & Minerals" opens a window where users can select components to add to the claim search.

Load Specifications:
Clicking the "Load Specifications" button opens a window where users can enter an Item Code to automatically load claim information into the search fields.

Search Execution:
Clicking the "Search" button updates the product list based on the applied filters, while the "Clear" button resets search conditions.
A "Re-search" function allows users to refine results within a previous search.

7. Product Comparison

Comparison Function (Compare)

Users can select checkboxes for products on the main page and click the 'Compare' button to navigate to the comparison page.

At the top of the comparison page, entering an Item Code and clicking 'Add' adds the product to the comparison table.

Comparison Page Includes:

  • Basic Information Table: Displays each product's Item Code, Unit Size, Color, Weight, Dosage, and Remarks.
  • Claim Table: Compares claim data with results marked in different colors (Black: Test O, Red: Test X or Input).

Deleting Products: The "Delete" button allows users to remove products from the comparison list.
